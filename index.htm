Have your cake and eat it too!

My SVN checkouts and Git clones reside in the same folder on my local machine. Changes from one can be committed to the other. Now when I commit I can have a URL to a working preview instantly!

This page lives in a lot of places:

  • On GitHub: https://github.com/kingkool68/The-Great-Testing-Repository
  • On my SVN Repository: http://svn.kingkool68.com/projects/The-Great-Testing-Repository/
  • A dev version (updated on every commit to my SVN server): http://dev.kingkool68.com/The-Great-Testing-Repository/

Some Downsides

  • Need to add .svn to the Git Ignore file.
  • Need to ignore the .git folder in the SVN repository.
